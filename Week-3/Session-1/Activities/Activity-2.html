Activity 2: Create New Elements

Your Task:

Make the "Add Item" button work. When clicked, it should create a new paragraph with the text from the input.

Steps:

  1. Get the text from the input
  2. Create a new paragraph element
  3. Set the paragraph's text
  4. Add it to the list container

Code Help:

document.getElementById('text-input').value - Get input text

document.createElement('p') - Create paragraph

element.textContent = 'text' - Set text

container.appendChild(element) - Add to page
